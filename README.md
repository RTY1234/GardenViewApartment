# Garden View Apartment - Guest Information

A static guest information webpage for Garden View Apartment in Cēsis, Latvia. This page provides essential information for guests who are currently staying at the property.

## Features

- 📱 Fully responsive design
- 📶 WiFi credentials clearly displayed
- 🔑 Check-in/check-out instructions
- 🏠 Apartment guide and appliance instructions
- 📋 House rules and important information
- 🍽️ Local restaurant and attraction recommendations
- 📞 Emergency and host contact information
- 🅿️ Parking details

## Deployment to GitHub Pages

### Option 1: Create a new repository

1. Create a new repository on GitHub
2. Upload `index.html` to the repository
3. Go to repository Settings → Pages
4. Under "Source", select the main branch
5. Click Save
6. Your site will be available at `https://yourusername.github.io/repository-name`

### Option 2: Use an existing repository

1. Add the `index.html` file to your repository
2. Follow steps 3-6 from Option 1

## Customization Guide

### Update Property Information

Edit the following sections in `index.html`:

- **Property Name & Address**: Update the header section with your property name and location
- **WiFi Credentials**: Update network name and password in the WiFi section
- **Lockbox Code**: Change the check-in lockbox code for security
- **Contact Information**: Update host phone number and email
- **Local Recommendations**: Customize restaurants, stores, and attractions for your area
- **House Rules**: Adjust rules according to your property policies
- **Apartment Guide**: Update appliance and facility information

### Important Security Notes

- Change the lockbox code after each guest or use dynamic codes
- Update WiFi password regularly
- Verify all contact phone numbers are correct
- Test all links before publishing
